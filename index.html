MineStats

Your Discord bot for Minecraft!

Features

Lookup any java server with /check java everywhere where external commands are allowed or where MineStats is added in. Available info: Host, IP, Port, MOTD, Online/Max Players, Version, Protocol.

Lookup any bedrock server with /check bedrock everywhere where external commands are allowed or where MineStats is added in. Available info: Host, IP, Port, MOTD, Online/Max Players, Version, Protocol.

Setup your own Downtime Detector with /detector setup which monitors your Minecraft Server every minute and sends out a message in a channel you specified to let you and/or your community know when the server is down.
